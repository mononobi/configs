# to add keyboard changer into windows startup:

# copy 'FarsiKeyboard' folder into a permanent location for example:
# C:\Users\{username}\AppData\Local\

# then copy 'FarsiKeyboard.exe' as a shortcut into:
# C:\Users\{username}\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

# not that you should set the compatibility mode of 'KeyChanger.exe' to
# windows xp sp3 for all users. otherwise, it can't be started.
